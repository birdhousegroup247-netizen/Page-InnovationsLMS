## 🌟 OVERVIEW

The TekyPro test system is now **fully flexible** and supports all reasonable use cases for Students, Instructors, and Admins. Every aspect can be configured to match your needs.

---

## ✅ WHAT STUDENTS CAN DO

### 1. **Practice Tests (Self-Generated)**
Students can create and take practice tests anytime for self-assessment:
- ✅ Choose categories (or mix from all categories)
- ✅ Set difficulty distribution (easy/medium/hard counts)
- ✅ Configure time limit
- ✅ Use quick presets (Quick Quiz, Standard Test, etc.)
- ✅ Take unlimited practice tests
- ✅ **Always see results immediately** (with scores and explanations)
- ✅ Retake as many times as desired

**Navigation:** "Generate Test" → Configure → Take → View Results

---

### 2. **Assigned Tests (From Instructors/Admins)**
Students receive and complete tests assigned by instructors or admins:
- ✅ **View all assigned tests** in "My Assigned Tests"
- ✅ See test details:
  - Title, description
  - Number of questions
  - Time limit
  - Due date
  - Assigned by (instructor/admin name)
  - Attempts used / allowed
- ✅ **Status tracking:**
  - Not Started
  - In Progress
  - Completed
  - Overdue (if past due date)
  - Due Soon (within 3 days)
- ✅ **Start, continue, or retake tests** (if allowed)
- ✅ **View results** (only if instructor enabled "show results immediately")
- ✅ **Cannot start overdue tests** (unless already in progress)
- ✅ **See score and pass/fail status** (if results are visible)

**Navigation:** "My Assigned Tests" → Select Test → Start/Continue → Submit → View Results (if allowed)

---

## ✅ WHAT INSTRUCTORS CAN DO

### 1. **Create & Assign Tests to Students**
Instructors have full control over test creation and assignment:
- ✅ Create tests for their courses
- ✅ **Assign to specific students** or entire courses
- ✅ Set all test configurations (see below)
- ✅ View student results and analytics
- ✅ Edit tests before publishing
- ✅ Save as draft or publish immediately

**Navigation:** "My Tests" → "Create Test" → Configure → Assign Students → Publish

---

### 2. **Contribute Questions to Question Bank**
Instructors can add questions for admin approval:
- ✅ Create questions (MCQ, True/False, Fill-in-Blank)
- ✅ Questions start as **"Pending"** status
- ✅ Admin reviews and approves/rejects
- ✅ **Approved questions available for ALL tests** (instructor, admin, practice)
- ✅ Can only edit pending questions (approved are locked)
- ✅ Track approval status

**Workflow:**
1. Instructor → "Contribute Questions" → Add Question
2. Admin → "Question Bank" → Review → Approve
3. Question now available for all tests

**Navigation:** "Contribute Questions" → Add → Wait for Admin Approval

---

## ✅ WHAT ADMINS CAN DO

### 1. **Full Question Bank Management**
- ✅ Create/Edit/Delete all questions
- ✅ Approve/Reject instructor-contributed questions
- ✅ Bulk import questions via CSV
- ✅ Bulk approve/delete operations
- ✅ Filter by category, difficulty, type, approval status
- ✅ Search questions

### 2. **Advanced Test Creation**
- ✅ Same capabilities as instructors

- ✅ Plus: Access to all questions (including pending ones)

## 🎛️ TEST CONFIGURATION OPTIONS (Instructors & Admins)

### **Basic Settings**
| Setting | Description | Example |
|---------|-------------|---------|
| **Title** | Test name | "Midterm Exam - Python Basics" |
| **Description** | Test details | "Covers modules 1-5" |
| **Course** | Link to course | "Python for Beginners" |
| **Due Date** | Submission deadline | "Dec 31, 2025 11:59 PM" |

---

### **Question Selection**
| Method | Description |
|--------|-------------|
| **Manual Selection** | Choose specific questions from question bank |
| **Auto-Generate** | Set difficulty distribution (e.g., 10 easy, 15 medium, 5 hard) |
| **Mix Both** | Manually select some, auto-generate others |

---

### **Time & Scoring**
| Setting | Options | Impact |
|---------|---------|--------|
| **Time Limit** | 15 min - Unlimited | Auto-submit when time expires |
| **Passing Score** | 0% - 100% | Determines pass/fail status |
| **Total Marks** | Auto-calculated | Sum of all question marks |

---

### **Question Behavior**
| Setting | Options | Impact |
|---------|---------|--------|
| **Shuffle Questions** | Yes / No | Randomize question order for each student |
| **Shuffle Options** | Yes / No | Randomize MCQ answer options |

---

### **Attempt Control**
| Setting | Options | Impact |
|---------|---------|--------|
| **Max Attempts** | 1, 2, 3, Unlimited | How many times students can retake |
| **Retakes Allowed** | Based on max attempts | Students see "Retake" button if under limit |

---

### **🔐 RESULTS VISIBILITY (KEY FLEXIBILITY)**

This is where **maximum flexibility** comes in:

| Setting | Options | Student Experience |
|---------|---------|-------------------|
| **Show Results Immediately** | ✅ Yes | Students see score, pass/fail, and review immediately after submission |
| **Show Results Immediately** | ❌ No | Students see "Test Submitted Successfully!" message. Results hidden until instructor releases them. |
| **Show Explanations** | ✅ Yes | Students see correct answers and explanations for all questions |
| **Show Explanations** | ❌ No | Students only see their score and pass/fail status, NOT correct answers or explanations |

---

### **🎯 FLEXIBILITY SCENARIOS**

#### **Scenario 1: Practice Quiz (Immediate Feedback)**
```
✅ Show Results Immediately: Yes
✅ Show Explanations: Yes
✅ Max Attempts: Unlimited
✅ Time Limit: 15 minutes

Result: Students see everything immediately, can retake unlimited times.
Perfect for practice and learning.
```

#### **Scenario 2: Graded Exam (Delayed Results)**
```
❌ Show Results Immediately: No
❌ Show Explanations: No
✅ Max Attempts: 1
✅ Time Limit: 120 minutes
✅ Due Date: End of semester

Result: Students submit exam, see "Test Submitted Successfully!"
Instructor reviews, then manually releases results later.
Perfect for high-stakes exams.
```

#### **Scenario 3: Self-Assessment (Show Score Only)**
```
✅ Show Results Immediately: Yes
❌ Show Explanations: No
✅ Max Attempts: 2
✅ Time Limit: 30 minutes

Result: Students see their score and pass/fail immediately,
but NOT the correct answers. Encourages them to study more.
Can retake once.
```

#### **Scenario 4: Homework (Full Transparency)**
```
✅ Show Results Immediately: Yes
✅ Show Explanations: Yes
✅ Max Attempts: 3
✅ Time Limit: None
✅ Due Date: Next week

Result: Students can take their time, see all answers and explanations,
retake up to 3 times to improve their score. Learning-focused.
```

## 🔄 COMPLETE WORKFLOWS

### **Practice Test Workflow (Student)**
```
1. Student → "Generate Test"
2. Configure test (categories, difficulty, time)
3. Click "Generate & Start Test"
4. Answer questions
5. Submit test
6. View results IMMEDIATELY (always)
7. See score, correct answers, explanations
8. Click "Take Another Test" to repeat
```

---

### **Assigned Test Workflow (Instructor → Student)**

**Instructor Side:**
```
1. Instructor → "My Tests" → "Create Test"
2. Add basic info (title, description, course, due date)
3. Select questions (manual or auto-generate)
4. Configure settings:
   - Time limit: 60 minutes
   - Max attempts: 2
   - Passing score: 70%
   - Show results immediately: Yes
   - Show explanations: No (students see score but not answers)
   - Shuffle questions: Yes
   - Shuffle options: Yes
5. Assign to students (select individuals or entire course)
6. Publish test
```

**Student Side:**
```
1. Student logs in
2. Sees notification: "New test assigned: Midterm Exam"
3. Student → "My Assigned Tests"
4. Sees test with status: "Not Started" + "Due Soon" badge
5. Clicks "Start Test"
6. Completes test within 60 minutes
7. Submits test
8. Sees score: 85% (Passed) ← Because "show results immediately" is ON
9. Reviews questions, sees selected answers
10. Does NOT see correct answers ← Because "show explanations" is OFF
11. Can click "Retake" ← Because attempted 1/2 times
12. Takes test again, scores 92%
13. Cannot retake again ← Reached max attempts (2/2)
```

---

### **Question Contribution Workflow (Instructor → Admin)**

**Instructor Side:**
```
1. Instructor → "Contribute Questions"
2. Click "Add Question"
3. Fill in details:
   - Question text: "What is the capital of France?"
   - Type: Multiple Choice
   - Options: Paris, London, Berlin, Rome
   - Correct answer: Paris
   - Difficulty: Easy
   - Category: Geography
   - Explanation: "Paris is the capital and largest city of France."
4. Save question
5. Question status: "Pending Review" (yellow badge)
6. Instructor can edit while pending
```

**Admin Side:**
```
1. Admin → "Question Bank"
2. Filter: "Pending" questions
3. Reviews question contributed by instructor
4. Clicks "Approve"
5. Question status changes to "Approved"
```

**Result:**
```
✅ Question is now available for:
   - Admin test creation
   - Instructor test creation
   - Student practice test generation
✅ Instructor can no longer edit (approved questions are locked)
✅ Instructor sees "Approved" badge in their contributions
```
